HOW TO BUILD THE EXE
================================================================================

METHOD 1: GITHUB ACTIONS (RECOMMENDED FOR MOST USERS)
------------------------------------------------------
1. Go to repository on GitHub
2. Click "Actions" tab
3. Select "Build Executable" workflow
4. Click "Run workflow"
5. Wait 20-30 minutes
6. Download artifact from bottom of page

No Python, no Visual Studio, no setup needed!

METHOD 2: PYINSTALLER (FOR DEVELOPERS)
---------------------------------------
Requirements: Python 3.8 or higher

Commands:
  pip install pyinstaller
  python build_pyinstaller.py --full --clean

Or on Windows:
  build_pyinstaller.bat

Output: dist/FaceAttendanceSystem_Package/

METHOD 3: NUITKA (FOR PRODUCTION)
----------------------------------
Requirements: Python 3.8.x, Visual Studio Build Tools, Windows

Commands:
  build.bat

Or:
  python build.py --full --clean

Output: dist/FaceAttendanceSystem_Package/
